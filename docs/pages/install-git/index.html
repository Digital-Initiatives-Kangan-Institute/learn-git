Installing Git

1. 🪟 Windows

  1. Open your web browser.
  2. Go to the official Git website: https://git-scm.com
  3. Click Download for Windows.
  4. Run the downloaded installer.
  5. Follow the installer steps and leave the default options selected.
  6. Click Install.

Check the installation

Open Command Prompt or PowerShell and run:

git --version

If Git is installed correctly, it will display the installed version.

2. 🐧 Linux

Git can be installed using your distribution’s package manager.

Debian / Ubuntu (APT)

sudo apt update
sudo apt install git

Fedora / RHEL (DNF)

sudo dnf install git

Arch Linux / Manjaro (Pacman)

sudo pacman -S git

Check the installation

git --version

3. 🍎 macOS

The easiest way is to install git is using Homebrew.

  1. Install Homebrew from the official website: https://brew.sh/
  2. Run the following command in a terminal:
brew install git

Check the installation

git --version
